# Typography

Work in progress!

**Primary** font is used for titles and **Secondary** font is used for paragraphs and small texts.

Typography is styled via css classes having semantic naming: `.text-{fontFamily}-{textType}-{weight}`

`fontFamily` is `primary` or `secondary`

`textType` is one of `display1`, `display2`, `title1`, `title2`, `text1`, `text2`, `caption1`, `caption2`

`weight` is one of `regular`, `medium`, `bold`, `extrabold`

NB: Not all combinations are allowed. Use only css classes that are presented here.
